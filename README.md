# ⚡ The Last-Minute Life Saver

An AI-powered productivity companion that proactively helps users plan, prioritize, and complete tasks before deadlines are missed.

## 🚀 Live Demo
[Open App](https://last-minute-life-saver-396925592411.asia-southeast1.run.app)

## ✨ Features
- 🏠 **Command Center** — Dashboard with live stats
- ✅ **Task Manager** — Add, prioritize, track tasks
- 🔥 **Habit Tracker** — Daily habits with streak system
- 📅 **Timeline** — Visual deadline view with overdue alerts
- 🤖 **AI Assistant** — Gemini-powered chat for scheduling & tips

## 🛠 Technologies Used
- HTML, CSS, JavaScript
- Gemini 1.5 Flash API
- Google AI Studio (Build & Deploy)
- Google Cloud Run
